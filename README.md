# TikTok Region Battle – დაყენების ინსტრუქცია

## საჭირო პროგრამები
- **Node.js** (ვერსია 16+) → https://nodejs.org
- **Git** (სურვილისამებრ)

---

## 1. დაყენება

```bash
# 1. გახსენი Terminal (ბრძანების ხაზი)
# 2. გადი პროექტის საქაღალდეში
cd tiktok-region-battle

# 3. დააინსტალირე ბიბლიოთეკები
npm install
```

---

## 2. გაშვება

```bash
# ჩვეულებრივი გაშვება
node server.js

# ან, თუ გინდა ავტო-გადატვირთვა ცვლილებებზე (dev)
npx nodemon server.js
```

სერვერი გაეშვება: **http://localhost:3000**

---

## 3. TikTok-ის მოთხოვნა

**მნიშვნელოვანი:** TikTok კავშირი მუშაობს ᲛᲮᲝᲚᲝᲓ:
- როდესაც ექაუნთი **LIVE-შია** TikTok-ზე
- ექაუნთს უნდა ქონდეს სულ მცირე 1000 ფოლოუერი (TikTok-ის წესი)

### ადმინ პანელში:
1. `@username` ველში ჩაწერე TikTok ექაუნთის სახელი (@ გარეშეც მუშაობს)
2. განსაზღვრე რეგიონები (შეგიძლია შეცვალო, წაშალო, დაამატო)
3. დააყენე ხანგრძლივობა
4. დააჭირე **"თამაშის დაწყება"**

---

## 4. როგორ მუშაობს

```
viewer-ი LIVE-ში წერს კომენტარს: "კახეთი"
    ↓
სერვერი ინახავს: user123 → კახეთი

viewer-ი აგზავნის გიფტს (100 ქოინი)
    ↓
სერვერი: კახეთი += 100,  user123 += 100

MVP ვინც ყველაზე მეტი ქოინი გაგზავნა
```

**ლოგიკა:** viewer-მა ჯერ კომენტარში უნდა ჩაწეროს რეგიონი,  
შემდეგ მისი გიფტები ამ რეგიონს ემატება.
